### Added
- Initial release of GISMOL toolkit.
- Core COHObject, constraints, daemons, and repository.
- Neural components: Regressor, Classifier, Generator, AnomalyDetector.
- NLP module: dialogue manager, intent recognition, constraint parser.
- Reasoners: BaseReasoner, TemporalReasoner, SafetyReasoner, and all domain‑specific reasoners.
- Five complete domain examples (supply chain, education, manufacturing, space, social welfare).
- Full API documentation and developer tutorial.
